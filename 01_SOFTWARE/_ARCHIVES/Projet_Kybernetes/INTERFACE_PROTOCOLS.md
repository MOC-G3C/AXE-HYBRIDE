# Interface Protocols
**System:** Kybernetes-Link
**Language:** English (Standard) / French (Legacy support)

## 1. Interaction Modes
* **Standard Mode:** Balanced detail. Default setting.
* **High-Bandwidth Mode:** Concise, dense information, code-heavy. Trigger: "Mode Dev".
* **Deep Dive Mode:** Explanatory, educational, philosophical. Trigger: "Explain concepts".

## 2. Universal Commands
* `/stop` - Immediate halt of generation.
* `/reset` - Clear current context window focus.
* `/save` - Reminder to commit to GitHub.

## 3. Formatting Standards
* **Code:** Must use triple backticks with language specified (e.g., `bash`, `python`).
* **Updates:** Use standard changelog format (Added, Changed, Fixed).
* **Emphasis:** Bold for key entities, Italic for metadata.
