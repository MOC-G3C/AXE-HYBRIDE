# UNIVERSAL DIRECTORY STANDARDS

To maintain low entropy across the Hybrid Axis, all sub-projects must adhere to this structure:

## Standard Folder Hierarchy

### /00_Admin
* **Purpose:** Governance and Logic.
* **Contains:** Licenses, Readme, To-Do lists, changelogs.

### /01_Core
* **Purpose:** The Engine.
* **Contains:** Source code, scripts, executables, main logic files.

### /02_Data
* **Purpose:** Memory.
* **Contains:** Inputs, datasets, generated logs, output results.

### /03_Research
* **Purpose:** Exploration.
* **Contains:** Notes, drafts, theoretical concepts, reference PDFs.

---
*Rule:* If a file does not fit these categories, it must be processed or discarded. Clutter is prohibited.
